**Tales** is a declarative integration and end-to-end testing tool, packaged as a single Go binary. You write tests as plain `.tales` files using a readable HCL2 syntax, and a single `tales test` command runs them with deterministic, seedable data generation.

This page is the entry point to the Tales documentation. If you are new, read the rest of this section in order: [Installation](/tales/docs/getting-started/installation/), [Your first scenario](/tales/docs/getting-started/first-scenario/), and [CLI essentials](/tales/docs/getting-started/cli-essentials/).
